# ML Age Predictor Website

A website documenting our machine learning project that predicts human age from facial images using a ResNet18 deep learning model.

## Overview

This is a Next.js website that showcases our ML age prediction project, including information about the dataset, methodology, and results.

## Running the Website

### Prerequisites
- Node.js 18+ and npm

### Steps

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser
